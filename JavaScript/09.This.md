# This

- 일반(Normal) 함수는 호출 위치에서 따라 this 정의
- 화살표 함수는 자신이 선언된 함수 범위에서 this 정의
- 아래 코드를 작성해서 확인 해보면 hail.normal()은 정상적으로 나오지만 hail.arrow()는 undefined로 나오게 된다.

```javascript
const hail = {
  name: "Hail",
  normal: function () {
    console.log(this.name);
  },
  arrow: () => {
    console.log(this.name);
  },
};

hail.normal();
hail.arrow();
```
