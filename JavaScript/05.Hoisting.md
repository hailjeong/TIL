# 호이스팅(Hoisting): 함수 선언부가 유효범위 최상단으로 끌어올려지는 현상

- 아래 코드중 첫번째 코드를 실행시키면 에러가 나오게 된다. 왜나하면 double 이라는 함수가 아직 만들어지지 않았기 때문에 콘솔창에 double은 함수가 아니다 라는 에러가 발생한다.
- 두번째 코드는 호이스팅이 발생해서 에러가 나오지 않게 된다.
- 
```javascript
const a = 7;

double();

const double = function () {
  console.log(a * 2);
};

const a = 7;

double();

function double() {
  console.log(a * 2);
}

```
